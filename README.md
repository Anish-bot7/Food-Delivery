# 🍅 Tomato - Food Delivery Application

Tomato is a modern full-stack food delivery web application that allows users to browse food items, add them to a cart, place orders, make payments, and track their order status in real time.

## 🚀 Features

### 👤 User Features

- Browse food menu
- Filter food by category
- Add and remove items from cart
- View cart summary
- Secure checkout process
- Place food orders
- Online payment integration
- Order history tracking
- Real-time order status updates

### 🍽️ Food Menu

- Explore multiple food categories
- View food images and descriptions
- Display pricing information
- Responsive food cards

### 🛒 Cart Management

- Add items to cart
- Increase/decrease quantity
- Remove items from cart
- Automatic total calculation
- Delivery fee calculation

### 💳 Order & Payment

- Delivery address collection
- Checkout page
- Payment gateway integration
- Payment verification system
- Order confirmation

### 📦 Order Tracking

- View previous orders
- Track order status
- View ordered items
- Order amount details

## 🔄 Application Workflow

1. User visits the website.
2. User explores food categories.
3. User adds food items to cart.
4. User proceeds to checkout.
5. Delivery information is collected.
6. Payment is processed through Stripe.
7. Order is created in the database.
8. User can track order status from "My Orders".
